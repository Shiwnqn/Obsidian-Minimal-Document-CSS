# Minimal Document

An Obsidian CSS snippet inspired by Minimal theme, focused on elegant document-style reading, refined spacing, and clearer visual hierarchy.

## Overview

Minimal Document is an Obsidian CSS snippet designed for a more editorial and document-like reading experience.

It keeps the restrained grayscale character of the Minimal theme, while improving visual hierarchy through typography, spacing, layout rhythm, table styling, list styling, and subtle surface contrast.

Instead of relying on colorful decoration, it focuses on structure, calmness, and long-form readability, making notes feel closer to a carefully designed document.

## Design Goals

- Keep a minimal, low-saturation visual style
- Extend the grayscale and blue-gray palette inspired by Minimal
- Improve recognition of headings, lists, tables, quotes, and code blocks
- Create better spacing and rhythm for long-form reading and writing
- Make notes feel more like polished documents than component-heavy interfaces

## Features

- Minimal-inspired grayscale palette with restrained accent color
- Refined document-style layout, spacing, and reading rhythm
- Clearer heading hierarchy and section structure
- Distinct styles for unordered lists, ordered lists, and task lists
- More polished tables, blockquotes, callouts, code blocks, and inline code
- Unified styling for links, tags, images, and embeds
- Better suited for long-term writing, reading, and note organization

## Best For

- Reading notes
- Literature notes
- Research notes
- Long-form writing
- Knowledge management vaults
- Users who like Minimal's palette but want a more document-oriented layout

## Installation

1. Copy `snippets/minimal-document.css` into your Obsidian vault's `.obsidian/snippets/` folder.
2. Open Obsidian.
3. Go to `Settings -> Appearance -> CSS snippets`.
4. Enable `minimal-document`.

## Notes

- Recommended for use with Obsidian's default theme.
- The palette is inspired by Minimal, while the layout and element styling are customized for a more refined document feel.
- If you prefer restrained, clean, and durable visual design, this snippet should fit well.
