# Assets

这个目录用于存放仓库展示截图。

推荐后续补充这些图片：

- `preview-reading.png`
  说明：阅读模式整体效果

- `preview-editor.png`
  说明：编辑模式整体效果

- `preview-table-list.png`
  说明：表格、列表、任务列表展示

- `preview-longform.png`
  说明：长文排版与标题层级展示

建议截图时：

- 使用 Obsidian 默认主题
- 启用 `minimal-document.css`
- 关闭无关插件面板或尽量减少干扰元素
- 保持窗口宽度统一，这样多张截图更整洁
