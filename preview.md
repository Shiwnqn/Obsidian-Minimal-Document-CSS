## 行内代码与代码块

你可以通过 `minimal-document.css` 来启用这套样式。

```css
.theme-light {
  --base-h: 0;
  --base-s: 0%;
  --base-l: 96%;
}
```

## 长段落示例

一个偏文档风格的样式，不一定依靠夸张的视觉差异来吸引注意力，而是通过稳定的版心、适度的留白、清晰的层级和统一的细节语言，让内容本身更容易被阅读和理解。对于需要长期记录、阅读、整理和写作的用户来说，这种设计通常会更耐看，也更容易在使用中保持舒适感。
